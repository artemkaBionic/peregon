Model: ASUS GL553V

Details

  • Power on the computer and repeatedly press the F2 key until the BIOS Utility appears.
  • Press the F9 key to Load Optimized Defaults.
  • Select OK and press ENTER to confirm.
  • Press the F7 key to view advanced options.
  • Navigate to Advanced using the RIGHT ARROW key.
  • Select Network Stack Configuration using the DOWN ARROW key and press ENTER.
  • Navigate to Network stack using the DOWN ARROW key and press ENTER.
  • Select Enabled using the DOWN ARROW key and press ENTER.
  • Press the F10 key to save configuration and exit.
  • Select OK and press ENTER to confirm.
  • While rebooting, repeatedly press the ESC key until the boot menu appears.
  • Select UEFI: IP4 Realtek PCIe GBE Family Controller using the DOWN ARROW key and press ENTER.
  • When the Refresh Environment has loaded, follow the instructions to start the refresh.
  • After the refresh is finished, click the OK button to turn off the computer.
