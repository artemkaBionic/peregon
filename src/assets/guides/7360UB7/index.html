Model: ASUS R704A

Details

  • Power on the computer and repeatedly press the F2 key until Setup is open.
  • Press the F9 key to load setup defaults.
  • Select YES and press ENTER to confirm selection.
  • Using the RIGHT ARROW key, navigate to the Advanced settings.
  • Press the DOWN ARROW key until Network Stack is selected and press ENTER.
  • Press ENTER and select ENABLED using the DOWN ARROW key and press ENTER.
  • Check that Ipv4 PXE Support is ENABLED.
  • Press the F10 key to save changes and exit.
  • Select Yes and press ENTER to confirm.
  • While rebooting, repeatedly press the ESC key until the "Please select boot device" menu appears.
  • Select "UEFI: Atheros Boot Agent" and press ENTER.
  • When the Refresh Environment has loaded, follow the instructions to start the refresh.
  • After the refresh is finished, click the OK button to turn off the computer.
