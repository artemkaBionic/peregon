Follow this steps to refresh Mac:

  • Insert USB drive into Mac. We highly recommend to charge Mac while refreshing so if Mac has only one USB-C drive then we recommend to use splitter as shown on image.
  • Power on (or reboot) Mac and hold the Option key until boot options are visible and select ‘Refresh’.
  • When the Refresh process is complete, hold the power button down until the shutdown options are displayed and click ‘Shut Down’.
