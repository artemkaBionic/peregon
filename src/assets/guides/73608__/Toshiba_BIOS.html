73608xx

Model: Toshiba(Windows 7 and earlier)

Details

  • While holding the F2 key, power on the computer.
  • When "Press <F2> to enter SETUP" appears, release the F2 key.
  • Press the F9 key to load setup defaults.
  • Select YES and press ENTER to confirm selection.
  • Press F10 and then ENTER to save and close the BIOS configuration.
  • While rebooting, hold the F12 key.
  • When the BOOT MENU appears, release the F12 key.
  • Select 5. LAN using the DOWN ARROW key and press ENTER.
  • When the Refresh Environment has loaded, follow the instructions to start the refresh.
  • After the refresh is finished, click the OK button to turn off the computer.
