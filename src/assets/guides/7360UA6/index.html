7360UA6

Model: ASUS

Details

  • Power on the computer and repeatedly press the F2 key until "F2 Setup" appears.
  • Press the F9 key to load setup defaults.
  • Select YES and press ENTER to confirm selection.
  • Navigate to ADVANCED using the RIGHT ARROW key.
  • Choose NETWORK STACK using the DOWN ARROW key and press ENTER.
  • Press ENTER and select ENABLED using the DOWN ARROW key and press ENTER.
  • Check that Ipv4 PXE Support is ENABLED.
  • Press F10 and then ENTER to save and close the BIOS configuration.
  • While rebooting, repeatedly press the ESC key until "CHOOSE SELECT BOOT DEVICE" appears.
  • Select "PXE" as boot option and press ENTER.
  • When the Refresh Environment has loaded, follow the instructions to start the refresh.
  • After the refresh is finished, click the OK button to turn off the computer.
