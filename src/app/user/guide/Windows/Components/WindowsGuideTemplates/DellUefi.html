Follow these steps to refresh DELL (Windows 8 and later):

1. Power on the computer and repeatedly press the F2 key until the Setup Utility appears.

2. Press the F9 key to load setup defaults, then select YES and press ENTER to confirm.

3. Navigate to Boot, select Boot List Option, and press ENTER.

4. Select UEFI and press ENTER.

5. Select Load Legacy Option Rom, and press ENTER.

6. Select Disabled and press ENTER.

7. Select Secure Boot, and press ENTER.

8. Select Enabled and press ENTER.

9. Press the F10 key to save changes and exit, then select Yes and press ENTER to confirm.

10. While rebooting, repeatedly press the F12 key until the Boot Manager appears.

11. Select "UEFI Onboard LAN IPv4" and press ENTER.

12. When the Refresh Environment has loaded, follow the instructions to start the refresh.

13. After the refresh is finished, click the OK button to turn off the computer.
