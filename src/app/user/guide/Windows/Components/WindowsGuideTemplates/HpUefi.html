Follow these steps to refresh HP (Windows 8 and later):

1. Power on the computer and repeatedly press the F10 key until "F10...BIOS Setup Options" appears.

2. Press the F9 key to load setup defaults, then select YES and press ENTER to confirm.

3. Navigate to System Configuration, select Boot Options, and press ENTER.

4. Choose Internal Network Adapter Boot and press ENTER.

5. Select EnabledÂ and press ENTER.

6. Press the F10 key to save changes and exit, then select Yes and press ENTER to confirm.

7. While rebooting, repeatedly press the F12 key until "F12...Network Boot" or "Please select boot device" appears.

8. If prompted to "Please select boot device", select "UEFI: IPv4 ..." and press ENTER.

9. When the Refresh Environment has loaded, follow the instructions to start the refresh.

10. After the refresh is finished, click the OK button to turn off the computer.
