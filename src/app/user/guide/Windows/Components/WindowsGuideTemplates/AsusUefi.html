Follow these steps to refresh ASUS (Windows 8 and later):

1. Power on the computer and repeatedly press the F2 key until Setup is open.

2. Press the F9 key to load setup defaults, then select YES and press ENTER to confirm.

3. Navigate to Advanced settings, select Network Stack, and press ENTER.

4. Press ENTER, select Enabled, and press ENTER. Check that Ipv4 PXE Support is enabled.

6. Press the F10 key to save changes and exit, then select Yes and press ENTER to confirm.

7. While rebooting, repeatedly press the ESC key until the "Please select boot device" menu appears.

8. Select "UEFI: Atheros Boot Agent" and press ENTER.

9. When the Refresh Environment has loaded, follow the instructions to start the refresh.

10. After the refresh is finished, click the OK button to turn off the computer.
