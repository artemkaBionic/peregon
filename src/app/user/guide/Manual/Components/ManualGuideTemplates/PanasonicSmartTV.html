Is refresh successful?

Follow these steps to refresh Panasonic TV:

1. Press Menu on the remote.

2. Press [▲ or ▼] to select Setup then press OK.

3. Press [▲ or ▼] to select System then press OK.

4. Press [▲ or ▼] to select Factory default then press OK.

5. Press [► or ◄] to select Yes then press OK.

6. It takes a few minutes to reset. While reset is in progress refrain from any activity. Do not turn off or unplug the TV.

7. If the TV reboots with factory settings, then refresh was successful.
