Is refresh successful?

Follow these steps to refresh LG Blu-Ray disc player:

1. Turn on LG Blu-Ray disc player.

2. Connect the LG Blu-Ray disc player to a TV so you can see the Blu-Ray disc player menu.

3. Using the Blu-Ray disc player remote select Settings and then select Others.

4. Select Initialize and then Factory Set to reset the player to the original factory settings.

5. If the blu-ray player reboots with factory settings, then refresh was successful.
