Is refresh successful?

Follow these steps to refresh Xbox 360:

1. Turn on Xbox 360.

2. Go to My Xbox/System settings.

3. In System settings go to Console settings/System info and write down the system serial number.

4. Go back to System settings and select Memory.

5. Under Memory press the 'Y' yellow button on your controller for device options.

6. Select Format (This will remove/delete everything) and select 'Yes'.

7. Enter your serial number, select done-confirm the serial number and select 'Yes'.

8. Wait until refresh will be completed.

9. If the Xbox 360 reboots with factory settings, then refresh was successful.
