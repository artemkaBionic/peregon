Model: HP

Details

  • Power on the computer and repeatedly press the F10 key until "F10...BIOS Setup Options" appears.
  • Select Apply Defaults and Exit using the DOWN ARROW key and press ENTER.
  • Select YES and press ENTER to confirm selection. Computer will reboot automatically.
  • While rebooting, repeatedly press the F10 key until "F10...BIOS Setup Options" appears.
  • Navigate to Security using the RIGHT ARROW key.
  • Select Network Boot using the DOWN ARROW key and press ENTER.
  • Select Enable using the RIGHT ARROW key.
  • Press F10 to accept selection.
  • Press F10 and then ENTER.
  • Select YES and press ENTER to save and close the BIOS configuration
  • While rebooting, repeatedly press the F12 key until "F12...Network Boot" appears.
  • When the Refresh Environment has loaded, follow the instructions to start the refresh.
  • After the refresh is finished, click the OK button to turn off the computer.
