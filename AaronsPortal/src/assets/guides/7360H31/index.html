Model: HP Pavilion 23 All-in-One PC 23-g218

Details

  • Power on the computer and repeatedly press the F10 key until the Setup Utility appears.
  • Select Apply Defaults and Exit using the DOWN ARROW key and press ENTER.
  • Select YES and press ENTER to confirm selection. Computer will reboot automatically.
  • While rebooting, repeatedly press the F12 key until "Please select boot device" appears.
  • Select UEFI: IPv4 Realtek PCIe GBE Family Controller using the DOWN ARROW key and press ENTER.
  • When the Refresh Environment has loaded, follow the instructions to start the refresh.
  • After the refresh is finished, click the OK button to turn off the computer.
