Model: Dell Inspiron 15 i3541-4000BLK

Details

  • Power on the computer and repeatedly press the F2 key until the System Setup is open.
  • Press the F9 key to load setup defaults.
  • Select YES and press ENTER to confirm.
  • Navigate to Boot using the RIGHT ARROW key.
  • Select Boot List Option using the DOWN ARROW key and press ENTER.
  • Choose UEFI using the DOWN ARROW key and press ENTER.
  • Select Secure Boot using the DOWN ARROW key and press ENTER.
  • Choose Enabled and press ENTER.
  • Press the F10 key to save changes and exit.
  • Select Yes and press ENTER to confirm.
  • While rebooting, repeatedly press the F12 key until the Boot Manager is open.
  • Select Onboard NIC(IPv4) using the DOWN ARROW key and press ENTER.
  • When the Refresh Environment has loaded, follow the instructions to start the refresh.
  • After the refresh is finished, click the OK button to turn off the computer.
