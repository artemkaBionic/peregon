7360DS2

Model: DELL

Details

  • Power on the computer and repeatedly press the F2 key until "F2 Setup" appears.
  • Navigate to EXIT using the RIGHT ARROW.
  • Select Load Default Setting using the DOWN ARROW.
  • Select YES and press ENTER to confirm selection.
  • Navigate to BOOT using the LEFT ARROW key.
  • Choose NETWORK DRIVES using the DOWN ARROW key and press ENTER.
  • Select 1ST DRIVE and Press ENTER
  • Select NETWORK…. and press ENTER.
  • Press F10 and then ENTER to save and close the BIOS configuration.
  • While rebooting, repeatedly press the F12 key until "F12...Network Boot" appears.
  • When the Refresh Environment has loaded, follow the instructions to start the refresh.
  • After the refresh is finished, click the OK button to turn off the computer.
