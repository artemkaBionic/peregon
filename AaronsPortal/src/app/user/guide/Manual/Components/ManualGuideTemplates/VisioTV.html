Follow these steps to refresh Visio TV:

1. Turn on Visio TV.

2. Press the VIA button on the remote control to bring up the VIA dock.

3. Select the HDTV SETTINGS from the VIA dock.

4. Select HELP/CLEAR MEMORY (FACTORY DEFAULT).

5. Enter '0000' for the Parental Control Passcode and select 'YES' and follow further instruction.

6. If the TV reboots with factory settings, then refresh was successful.
