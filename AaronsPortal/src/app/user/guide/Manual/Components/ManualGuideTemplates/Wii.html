Follow these steps to refresh Wii device:

1. Turn on Wii device.

2. From the main menu, choose the Wii OPTIONS button on the lower left side of the screen.

3. From the options page, locate and select Wii SETTINGS.

4. There are 3 Wii system settings pages, use your remote to click the arrow on the right side of the screen until you have reached the 3rd page.

5. On page 3 you will see 4 options. Select the bottom option FORMAT Wii SYSTEM MEMORY.

6. You will now see a series of confirmation screens asking are you sure you want to proceed with the format. On these screens select FORMAT.

7. After that you see that the Wii system is formatting the memory. This will take a few moments.

8. Once the system format is complete, you will be prompted to click OK to shut down the machine. Make sure to test a game verifying that it starts and plays properly before your GM inspects it, it is priced and put back on the sales floor.
